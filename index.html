CON ETIQUETA VIEWPORT

La etiqueta meta para el viewport fue introducida por Apple la creó para decirle al iPhone como debería renderizar el documento,

para ayudar a los desarrolladores a mejorar la presentación de sus aplicaciones web

en un iPhone, iPod Touch o iPad.

La etiqueta viewport permite que los sitios web o web apps,

definir el ancho, alto y escala del área usada

por el navegador para mostrar contenido.
